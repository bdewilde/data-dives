# data-dives

A series of explorations — "dives" of varying depth — into data sets, tasks, methods, and tools.

## installation

```shell
$ git clone git@github.com:bdewilde/data-dives.git
$ cd data-dives
$ pip install -e .
```

For development, also install the dev dependencies:

```shell
$ pip install -e .[dev]
```
